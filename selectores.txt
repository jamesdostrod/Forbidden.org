برای استایل به المنت ها
لمنت هارو سلکت و بهش استایل میدن
پنج نوع سلکتور داریم
سلکتور ها استایل دهی را راحت تر میکنند
سلکتور ها همیشه از بالا به پایین شروع به بررسی میکنند

1.simple selectore:
تا الان باهاشون کار کردیم
که عبارت است از اسم تگ، با آیدی، کلاس دهی، ستاره دهی که همه المنت های داخل داکیمنت را در نظر میگیرد

<body>
    <p><i>h1 as an example</i></p>
    <section id="header"></section>
    اینجا پی اولین فرزند بادی هست
    و سکشن دومین فرزند
    و اولین پرنتشون بادی هست
    و اولین پرنت آی یعنی ایتالیک پی هست

- #id{}
- .class{}
- element.classname p.wb
- element p{} body{}
- * {}
- div,p,h1,h2 {}

2.combinator element:
المنت ها را بر اساس ارتباط بین آنها انتخاب میکند

- element element - div p => همه پی هایی که داخل دیو قراردارد
- element>element - div>p => همه پی هایی که اولین پرنتشون دیو هست
- element + element - div +p => اولین پی که بلافاصله بعد از تمام شدن دیو قرار دارد
- element ~element - p ~ul => هر یو الی که بعد از اتمام پی قرار بگیرد را انتخاب انتخاب میکنه هر چند تا هم باشه آنها را انتخاب میکنه
و وسط اگر المنتی باشه مهم نیست
مثلا برای عکس های بین متون وبلاگ مورد استفاده هست که فقط به ایمیج استایل اعمال کنیم

3.pseudo-class selectores:
المنت ها را بر اساس حالت خاصی از آنها انتخاب میکند

- a:link - a:visited - a:hover - a:active / p:hover etc.
- element.classname:hover - p.class1:hover{} اون پی که اسم کلاسش کلاس یک هست، فقط به این اعمال میکنه / با بالایی ها میکس میکنیم
- element:first-child اون پی که اولین فرزند همه المنت هاست
- element:first-child element - p:first-child h1 {} اچ یک که پرنتش پی باشه و اون پی اولین فرزند از والدش باشه
- q:lang(no) {} - قوانین خاصی برای زبان های مختلف رو انتخاب میکنه
- input:checked{} - یعنی موقعی که چک یا تیک زدیم استایل هایی که دادیم را اعمال کنه
- input:disabled - اتریبیوت دیس ایبل داره
- input:enabled - اتریبیوت انیبل داره / دیس ایبل نداره
- :empty - p:empty{} **********
- :first-of-type - p:first-of-type  - اون پی را انتخاب میکنه که اولین پی از فرزندان باشه
مهم نیست قبل از پی چه المنت هایی وجود داشته باشه مهم اینه که از بالا به پایین اولین پی که میبینه را انتخاب کنه
- :focus - input:focus - بعد کلیک در داخل اینپوت استایل ظاهر میشه
